# 도메인 용어 사전

IntoTheHeaven 서비스에서 사용하는 주요 용어를 정리합니다.

## 사용자 관련

| 용어 | 설명 |
|------|------|
| 멤버 | 서비스에 가입한 사용자 |
| 교회 | 서비스에 등록된 교회 단위 |
| 교회 역할 | 멤버(일반), 리더, 관리자, 최고관리자의 4단계 역할 |
| 교적부 | 교회 소속 전체 멤버를 검색하고 조회하는 기능 |
| 편입 요청 | 교회에 가입을 신청하는 것. 관리자가 승인하면 교회 소속이 됨 |

## 소그룹 관련

| 용어 | 설명 |
|------|------|
| 소그룹 | 셀, 목장 등 교회 내 소모임 단위 |
| 소그룹 타입 | 일반(NORMAL)과 새가족(NEWCOMER) 두 가지 유형 |
| 소그룹 역할 | 리더, 서브리더, 멤버의 3단계 역할 |
| 새가족 | NEWCOMER 타입 소그룹에 소속된 멤버 |

## 모임 관련

| 용어 | 설명 |
|------|------|
| 모임 | 소그룹의 정기 또는 비정기 만남 기록 |
| 출석 | 예배 출석과 모임 출석을 각각 독립적으로 기록 |
| 나눔 (Story) | 모임에서 나눈 이야기 |
| 한 주 목표 (Goal) | 모임에서 세운 다음 주 목표 |
| 리더 코멘트 | 소그룹 리더가 모임에 남기는 코멘트 |
| 목회자 코멘트 | 관리자가 모임에 남기는 코멘트 |

## 기도 관련

| 용어 | 설명 |
|------|------|
| 기도제목 | 모임, 심방, 개인의 3가지 맥락에서 작성하는 기도 요청 |
| 응답됨 | 기도제목이 응답된 상태를 표시하는 것 |

## 돌봄 관련

| 용어 | 설명 |
|------|------|
| 심방 | 목회자가 가정이나 개인을 방문하여 기록하는 것. 관리자 전용 기능 |

## 교육 관련

| 용어 | 설명 |
|------|------|
| 교육 프로그램 | 소그룹별로 설정하는 교육 커리큘럼. 주차별로 진행 |
| 졸업 | 새가족 교육을 완료한 후 일반 소그룹으로 전환하는 것 |

## 커뮤니케이션 관련

| 용어 | 설명 |
|------|------|
| 메시지 | 멤버 간 1:1 메시지. 생일 축하 메시지와 일반 메시지가 있음 |
| 알림 | 시스템이 보내는 알림. 목회자 코멘트 등록, 소모임 정보 업데이트 등 |

## 기타

| 용어 | 설명 |
|------|------|
| 이벤트 | 교회 캘린더에 표시되는 일정 |
| 세례 상태 | 세례(BAPTIZED), 미세례(NOT_BAPTIZED), 유아세례(PAEDOBAPTISM)의 3가지 상태 |
