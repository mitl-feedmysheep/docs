# 소그룹 관리

> 소그룹을 생성하고, 멤버를 할당하거나 제외합니다.

## 화면 설명

소그룹의 생성, 수정, 삭제와 멤버 할당/제외를 관리하는 화면입니다.

## 주요 기능

### 소그룹 생성

새로운 소그룹을 만듭니다.

1. **소그룹 생성** 버튼을 누릅니다.
2. 다음 정보를 입력합니다.
   - **이름** (필수)
   - **설명**
   - **시작일 / 종료일**
   - **타입**: 일반(NORMAL) 또는 새가족(NEWCOMER)
   - **교육 프로그램** (선택): 새가족 소그룹 등에 교육 프로그램을 연결할 수 있습니다.
     - 프로그램명
     - 설명
     - 주차 수
3. **생성** 버튼을 누르면 소그룹이 만들어집니다.

### 소그룹 관리

기존 소그룹을 수정하거나 삭제합니다.

- **연도 필터**: 연도별로 소그룹을 필터링합니다.
- **검색**: 소그룹명으로 검색합니다.
- **이름 수정**: 소그룹명을 인라인으로 바로 수정할 수 있습니다.
- **삭제**: 소그룹을 삭제합니다.

### 멤버 할당

멤버를 소그룹에 배정합니다.

1. **멤버 검색**란에 이름을 입력합니다. (입력 후 300ms 후 자동 검색)
2. 검색 결과에서 할당할 **멤버를 여러 명 선택**합니다.
3. 할당할 **소그룹**을 선택합니다.
4. **역할**을 선택합니다.
   - 멤버 (MEMBER)
   - 서브리더 (SUB_LEADER)
   - 리더 (LEADER)
5. **할당** 버튼을 누르면 선택한 멤버들이 해당 소그룹에 배정됩니다.

### 멤버 제외

소그룹에서 멤버를 제외합니다.

1. **소그룹 필터**로 대상 소그룹을 선택합니다.
2. 해당 소그룹의 멤버 목록이 표시됩니다.
   - 이름, 성별, 생년월일, 전화번호, 역할
3. 제외할 멤버의 **제외** 버튼을 누릅니다.

## 알아두면 좋은 점

- 소그룹 타입은 생성 시에만 설정할 수 있습니다.
- 교육 프로그램은 소그룹 생성 시 선택적으로 연결합니다.
- 멤버 할당 시 여러 멤버를 한번에 선택하여 일괄 할당할 수 있습니다.
- 리더 역할은 소그룹당 1명을 권장합니다.
