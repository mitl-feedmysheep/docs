# 기도제목 관리

> 나의 기도제목, 소모임별 기도제목, 심방 기도제목을 통합 관리합니다.

## 화면 설명

3개의 탭으로 구분하여 기도제목을 관리합니다. 관리자 본인의 기도제목, 소그룹 모임의 기도제목, 심방에서 나온 기도제목을 각각 확인하고 관리할 수 있습니다.

## 주요 기능

### 나의 기도제목 탭

관리자 본인의 기도제목을 관리합니다.

- **추가**: 새로운 기도제목을 작성합니다.
- **필터**: 전체 / 기도 중 / 응답됨으로 필터링합니다.
- **목록**: 기도제목, 설명, 생성일이 표시됩니다.
- **응답 체크**: 기도제목의 응답 여부를 토글합니다.
- **편집**: 기도제목 내용을 수정합니다.
- **삭제**: 기도제목을 삭제합니다.

### 소모임별 기도제목 탭

각 소그룹 모임에서 나온 기도제목을 확인합니다.

- **소그룹 필터**: 특정 소그룹의 기도제목만 볼 수 있습니다.
- **연도/월 네비게이션**: 좌우 화살표로 월을 이동하여 조회합니다.
- **필터**: 전체 / 기도 중 / 응답됨으로 필터링합니다.
- **멤버별 그룹화**: 기도제목이 멤버별로 묶여 표시됩니다. 각 멤버의 역할도 함께 표시됩니다.

### 심방 기도제목 탭

심방에서 나온 기도제목을 확인합니다.

- **월별 네비게이션**: 좌우 화살표로 월을 이동하여 조회합니다.
- **필터**: 전체 / 기도 중 / 응답됨으로 필터링합니다.
- **심방별 그룹화**: 기도제목이 심방 단위로 묶여 표시됩니다. 심방 날짜와 장소가 함께 표시됩니다.
- **대상자별 기도제목**: 각 심방 대상자의 기도제목이 구분되어 표시됩니다.

## 알아두면 좋은 점

- 나의 기도제목은 관리자 본인만 사용하는 개인 기도 노트입니다.
- 소모임별 기도제목은 조회 전용입니다. 기도제목의 추가/수정은 웹앱의 모임 상세에서 수행합니다.
- 심방 기도제목은 심방 관리에서 기록한 내용이 연동됩니다.
