# 이벤트 관리

> 교회 캘린더에 표시되는 일정을 관리합니다.

## 화면 설명

교회 이벤트(일정)를 생성, 수정, 삭제합니다. 등록된 이벤트는 웹앱 홈 화면의 미니 캘린더에 표시됩니다.

## 주요 기능

### 월별 이벤트 조회

- 좌우 화살표로 월을 이동하여 해당 월의 이벤트를 조회합니다.
- 이벤트가 목록 형태로 표시됩니다.

### 이벤트 생성

새로운 교회 일정을 등록합니다.

1. **이벤트 추가** 버튼을 누릅니다.
2. 다음 정보를 입력합니다.
   - **제목** (필수)
   - **설명** (선택)
   - **시작일** (필수)
   - **종료일** (필수)
   - **시작 시간** (선택) - 하루 종일 이벤트인 경우 비워둡니다.
   - **종료 시간** (선택)
   - **장소** (선택)
3. **저장** 버튼을 눌러 이벤트를 생성합니다.

### 이벤트 수정

기존 이벤트를 선택하여 정보를 수정합니다.

### 이벤트 삭제

이벤트를 삭제합니다.

## 알아두면 좋은 점

- 등록된 이벤트는 웹앱 홈 화면의 미니 캘린더에 자동으로 표시됩니다.
- 시작/종료 시간을 비워두면 하루 종일(all-day) 이벤트로 표시됩니다.
- 여러 날에 걸친 이벤트는 시작일과 종료일을 다르게 설정합니다.
