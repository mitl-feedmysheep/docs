# 로그인

> 관리자 계정으로 로그인하고, 관리할 교회를 선택합니다.

## 화면 설명

어드민 대시보드에 접속하기 위해 관리자 권한이 있는 계정으로 로그인합니다. 여러 교회의 관리 권한을 가진 경우 관리할 교회를 선택합니다.

## 주요 기능

### 관리자 로그인

1. 이메일과 비밀번호를 입력합니다.
2. **로그인** 버튼을 누릅니다.
3. 관리자(ADMIN) 이상의 역할을 보유한 교회 목록이 표시됩니다.

### 관리 교회 선택

1. 관리 권한이 있는 교회 목록에서 관리할 교회를 선택합니다.
2. 선택한 교회의 대시보드로 진입합니다.
3. 이후 사이드바에서 교회를 전환할 수 있습니다.

## 알아두면 좋은 점

- 관리자(ADMIN) 또는 최고관리자(SUPER_ADMIN) 역할을 가진 멤버만 어드민에 로그인할 수 있습니다.
- 로그인 토큰은 90일간 유지됩니다. 90일이 지나면 다시 로그인해야 합니다.
- 토큰은 보안을 위해 HttpOnly 쿠키로 저장됩니다.
