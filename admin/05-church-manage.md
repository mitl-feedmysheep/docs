# 교회 관리

> 교회 편입 요청을 관리하고, 새로운 멤버 계정을 생성합니다.

## 화면 설명

교회 가입을 요청한 멤버의 승인/거부 처리, 편입 히스토리 조회, 관리자가 직접 계정을 생성하는 기능을 제공합니다.

## 주요 기능

### 편입 요청 관리

교회 가입 요청을 승인하거나 거부합니다.

1. 대기 중인 요청자 목록이 표시됩니다.
   - 이름, 이메일, 성별, 생년월일, 전화번호, 요청일
2. 각 요청에 대해 **승인** 또는 **거부** 버튼을 눌러 처리합니다.
3. 승인하면 해당 멤버가 교회 소속이 됩니다.

### 편입 히스토리

과거 편입 요청의 처리 내역을 확인합니다.

- **필터**: 승인 / 거부 상태로 필터링할 수 있습니다.
- 각 히스토리에는 다음 정보가 표시됩니다.
  - 요청자 정보
  - 처리자 (누가 승인/거부했는지)
  - 요청일
  - 처리일

### 계정 생성

관리자가 직접 멤버 계정을 생성합니다. 멤버가 직접 가입하기 어려운 경우에 사용합니다.

1. 다음 정보를 입력합니다.
   - **이름** (필수)
   - **이메일** (필수) - 중복 체크 버튼으로 확인
   - **비밀번호** (필수)
   - **성별**
   - **생년월일**
   - **전화번호**
   - **주소**
   - **직업**
   - **세례 여부**
   - **MBTI**
   - **자기소개**
2. **생성** 버튼을 누르면 계정이 생성됩니다.
3. 생성 완료 후 이메일과 비밀번호를 **복사**할 수 있는 버튼이 제공됩니다. 해당 멤버에게 전달하여 로그인하도록 안내합니다.

## 알아두면 좋은 점

- 계정 생성 시 이메일은 다른 멤버와 중복될 수 없습니다.
- 생성된 계정은 자동으로 해당 교회에 소속됩니다.
- 생성 후 이메일/비밀번호 복사 기능을 활용하여 멤버에게 안전하게 전달하세요.
