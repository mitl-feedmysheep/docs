# 소그룹

> 교회 소그룹 목록을 조회하고 상세 정보를 확인합니다.

## 화면 설명

교회에 등록된 소그룹을 검색하고, 각 소그룹의 멤버 구성과 모임 기록을 확인할 수 있습니다.

## 주요 기능

### 소그룹 검색

검색창에 소그룹명 또는 리더명을 입력하여 검색합니다.

### 소그룹 카드 목록

각 소그룹 카드에는 다음 정보가 표시됩니다.

- 소그룹 이름
- 리더 이름
- 멤버 수
- 생성일
- 상태

### 소그룹 상세

소그룹을 선택하면 상세 정보를 확인할 수 있습니다.

#### 셀원 목록

소그룹에 소속된 멤버 목록이 표시됩니다.

- 이름
- 전화번호
- 역할 배지 (리더 / 서브리더 / 멤버)

#### 모임 기록

해당 소그룹의 모임 기록이 표시됩니다.

- 모임 날짜
- 출석자 수 / 전체 인원
- 출석률 배지 (색상으로 출석률 구간 표시)

## 알아두면 좋은 점

- 소그룹 생성, 수정, 멤버 할당은 [소그룹 관리](06-group-manage.md) 메뉴에서 수행합니다.
- 이 화면은 조회 전용입니다.
