# 교적부

> 교회 소속 전체 멤버를 검색하고 상세 정보를 확인합니다.

## 화면 설명

교회에 등록된 모든 멤버를 검색하고, 각 멤버의 상세 정보와 기도제목 히스토리를 확인할 수 있습니다.

## 주요 기능

### 멤버 검색

검색창에 이름, 이메일, 전화번호를 입력하여 멤버를 검색합니다.

### 멤버 카드 목록

검색 결과가 카드 형태로 표시됩니다. 각 카드에는 다음 정보가 포함됩니다.

- 프로필 이미지
- 이름
- 이메일
- 전화번호
- 성별
- 생년월일
- 주소
- 직업
- 세례 여부
- MBTI
- 특이사항

### 멤버 상세 페이지

멤버를 선택하면 상세 페이지로 이동합니다.

#### 기본 정보

- 이메일, 전화번호, 생년월일, 주소
- 직업, 세례 여부, MBTI
- 소속 소그룹
- 특이사항
- 등록일

#### 기도제목 히스토리

해당 멤버의 기도제목 히스토리를 확인할 수 있습니다.

- 최근 50개의 기도제목이 표시됩니다.
- 각 기도제목의 날짜, 소속 소그룹, 응답 여부가 표시됩니다.

## 알아두면 좋은 점

- 교적부는 해당 교회 소속 멤버만 조회됩니다.
- 기도제목 히스토리를 통해 멤버의 기도 여정을 한눈에 파악할 수 있습니다.
