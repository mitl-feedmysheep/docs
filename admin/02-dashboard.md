# 대시보드

> 교회 현황을 한눈에 파악할 수 있는 메인 화면입니다.

## 화면 설명

교회의 주요 통계, 이번 달 생일자, 주차별 모임 현황을 확인할 수 있습니다.

## 주요 기능

### 통계 카드

교회 현황을 요약한 4개의 통계 카드가 상단에 표시됩니다.

- **전체 멤버 수**: 교회에 등록된 전체 멤버 수
- **활성 소그룹 수**: 현재 운영 중인 소그룹 수
- **예배 참석률**: 전체 멤버 대비 예배 출석 비율
- **소그룹 참석률**: 전체 멤버 대비 소그룹 모임 출석 비율

### 월간 생일 섹션

이번 달 생일자를 확인합니다.

- 좌우 화살표로 월을 이동하여 다른 달의 생일자도 확인할 수 있습니다.
- 각 생일자의 이름, 성별, 생일 날짜, 요일이 표시됩니다.
- 오늘이 생일인 멤버는 하이라이트로 강조됩니다.

### 주차별 모임 현황

소그룹 모임 현황을 주차별로 상세하게 확인합니다.

#### 기간 선택

- 연도, 월, 주차를 선택하여 해당 기간의 모임 현황을 조회합니다.

#### 모임 현황 테이블

테이블에 다음 정보가 표시됩니다.

| 항목 | 설명 |
|------|------|
| 소그룹명 | 소그룹 이름 |
| 생성 여부 | 해당 주차에 모임이 생성되었는지 여부 |
| 장소 | 모임 장소 |
| 예배 출석 | 출석 인원 / 전체 인원 + 출석률 |
| 소그룹 출석 | 출석 인원 / 전체 인원 + 출석률 |
| 특이사항 | 모임 특이사항 |

#### 모임 상세 (Sheet)

테이블에서 모임을 클릭하면 상세 정보가 Sheet로 열립니다.

- **리더 코멘트**: 소그룹 리더가 작성한 코멘트
- **목회자 코멘트**: 관리자가 코멘트를 작성하고 저장할 수 있습니다. 저장 시 소그룹 멤버에게 알림이 발송됩니다.
- **멤버별 상세**: 각 멤버의 예배 출석, 모임 출석, 나눔, 한 주 목표, 기도제목을 확인할 수 있습니다.

#### 엑셀 다운로드

주차별 모임 현황을 엑셀 파일로 다운로드할 수 있습니다.

## 알아두면 좋은 점

- 목회자 코멘트를 작성하면 해당 소그룹 멤버에게 알림이 발송됩니다.
- 엑셀 다운로드를 활용하면 보고서 작성에 편리합니다.
