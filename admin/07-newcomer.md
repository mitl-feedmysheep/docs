# 새가족 관리

> 새가족 소그룹의 교육 현황을 확인하고, 졸업 처리를 합니다.

## 화면 설명

새가족(NEWCOMER) 타입 소그룹의 멤버 현황과 교육 이수 상태를 관리합니다. 교육을 완료한 멤버의 졸업 처리를 수행합니다.

## 주요 기능

### 요약 통계

상단에 새가족 관련 요약 통계가 표시됩니다.

- **전체 새가족**: 새가족 소그룹에 소속된 전체 멤버 수
- **졸업자**: 교육을 완료하고 졸업한 멤버 수
- **진행중**: 현재 교육 진행 중인 멤버 수

### 연도 필터

연도를 선택하여 해당 연도의 새가족 현황을 조회합니다.

### 새가족 그룹별 현황

각 새가족 소그룹의 상세 현황이 표시됩니다.

- **그룹명**
- **교육 프로그램 정보**: 프로그램명, 전체 주차 수, 졸업자 수
- **멤버별 이수 주차**: 각 멤버가 몇 주차까지 교육을 이수했는지 표시됩니다.

### 졸업 처리

교육을 완료한 멤버의 졸업을 처리합니다.

1. **졸업 처리** 버튼을 누릅니다.
2. 졸업 대상 **소그룹**을 선택합니다.
3. 해당 소그룹의 **모임을 다중 선택**합니다.
4. **졸업** 버튼을 눌러 처리를 완료합니다.

## 알아두면 좋은 점

- 졸업 처리는 새가족 소그룹에서 일반 소그룹으로의 전환을 의미합니다.
- 졸업 처리 시 해당 멤버의 교육 이수 기록은 유지됩니다.
- 새가족 소그룹은 [소그룹 관리](06-group-manage.md)에서 타입을 "새가족"으로 선택하여 생성합니다.
