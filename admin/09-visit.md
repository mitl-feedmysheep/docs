# 심방 관리

> 목회자의 가정/개인 방문 기록을 관리합니다.

## 화면 설명

심방(목회자 방문) 기록을 생성, 수정, 삭제하고, 심방 대상 멤버를 관리합니다.

## 주요 기능

### 심방 목록

등록된 심방 기록이 목록으로 표시됩니다.

- 날짜
- 시간
- 장소
- 비용
- 메모

### 심방 생성

새로운 심방을 기록합니다.

1. **심방 추가** 버튼을 누릅니다.
2. 다음 정보를 입력합니다.
   - **날짜** (필수)
   - **시작 시간 / 종료 시간** (필수)
   - **장소** (필수)
   - **비용** (선택)
   - **메모** (선택)
3. **저장** 버튼을 눌러 심방을 생성합니다.

### 심방 수정

기존 심방 기록을 수정합니다.

- 심방을 선택하여 날짜, 시간, 장소, 비용, 메모를 수정할 수 있습니다.

### 심방 멤버 관리

심방 대상 멤버를 추가하거나 제거합니다.

- **멤버 추가**: 멤버 검색을 통해 심방 대상자를 추가합니다.
- **멤버 제거**: 심방 대상 목록에서 멤버를 제거합니다.

### 심방 삭제

심방 기록을 삭제합니다.

## 알아두면 좋은 점

- 심방은 관리자 전용 기능으로, 일반 멤버(웹앱)에서는 접근할 수 없습니다.
- 심방에서 기록한 기도제목은 [기도제목 관리](08-prayer-manage.md)의 심방 기도제목 탭에서 확인할 수 있습니다.
