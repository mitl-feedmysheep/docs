# 멤버 관리

경로: **운영 관리 > 멤버 관리** (`/manage/member`)

멤버 계정 정보를 수정하거나 비밀번호를 초기화할 수 있습니다.

---

## 유저 정보 수정

이름 또는 전화번호로 교회 소속 멤버를 검색한 뒤, 기본 정보를 수정합니다.

### 검색

1. "이름 또는 전화번호" 입력란에 검색어 입력
2. 검색 결과 목록에서 대상 멤버 클릭
3. 기존 정보가 폼에 자동으로 채워짐

### 수정 가능 필드

| 필드 | 필수 | 비고 |
|------|------|------|
| 이름 | O | 최대 20자 |
| 이메일 | O | 중복 불가 |
| 성별 | O | 남성/여성 |
| 생년월일 | O | |
| 전화번호 | O | 최대 20자 |
| 주소 | X | 최대 100자 |
| 직업 | X | 최대 100자 |
| 세례 여부 | X | 세례/미세례/유아세례 |
| MBTI | X | 최대 4자 |
| 특이사항 | X | 최대 100자 |

### 저장

- "저장" 버튼 클릭 시 수정 사항이 반영됩니다.
- 이메일이 다른 멤버와 중복되면 에러가 표시됩니다.

---

## 비밀번호 초기화

멤버의 비밀번호를 관리자가 직접 초기화합니다.

### 사용 방법

1. "비밀번호 초기화" 탭 선택
2. 이메일로 대상 멤버 검색
3. 새 비밀번호 입력 (8자 이상)
4. "비밀번호 초기화" 버튼 클릭
