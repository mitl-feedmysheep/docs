# 시스템 관리

> 최고관리자(Super Admin) 전용 기능입니다. 교회 생성, 권한 관리, 비밀번호 초기화를 수행합니다.

## 화면 설명

시스템 전반을 관리하는 최고관리자 전용 화면입니다. 새로운 교회를 등록하고, 멤버의 교회 역할을 관리하며, 비밀번호를 초기화할 수 있습니다.

## 주요 기능

### 교회 생성

서비스에 새로운 교회를 등록합니다.

1. **교회 생성** 버튼을 누릅니다.
2. 다음 정보를 입력합니다.
   - **교회명** (필수)
   - **위치** (필수)
   - **전화번호** (선택)
   - **홈페이지** (선택)
   - **설명** (선택)
3. **생성** 버튼을 눌러 교회를 등록합니다.

### 권한 관리

멤버의 교회 역할을 할당합니다.

1. **멤버 검색**란에 이름을 입력하여 대상 멤버를 찾습니다.
2. 역할을 할당할 **교회**를 선택합니다.
3. **역할**을 선택합니다.
   - 멤버 (MEMBER) - 일반 사용자
   - 리더 (LEADER) - 소그룹 리더
   - 관리자 (ADMIN) - 교회 관리자, 어드민 대시보드 접근 가능
   - 최고관리자 (SUPER_ADMIN) - 시스템 관리 기능 접근 가능
4. **할당** 버튼을 누릅니다.

### 비밀번호 초기화

멤버의 비밀번호를 초기화합니다. 멤버가 비밀번호를 잊어버리고 이메일 인증도 어려운 경우에 사용합니다.

1. **멤버 검색**란에 이름을 입력하여 대상 멤버를 찾습니다.
2. **비밀번호 초기화** 버튼을 누릅니다.
3. 초기화된 비밀번호가 표시됩니다. 해당 멤버에게 전달합니다.

## 알아두면 좋은 점

- 이 화면은 최고관리자(SUPER_ADMIN) 역할을 가진 멤버만 접근할 수 있습니다.
- 관리자(ADMIN) 역할을 부여하면 해당 멤버가 어드민 대시보드에 로그인할 수 있게 됩니다.
- 비밀번호 초기화는 보안에 민감한 기능이므로 본인 확인 후 신중하게 사용하세요.
