# 시작하기

> 회원가입부터 교회 등록까지, 서비스를 시작하는 방법을 안내합니다.

## 화면 설명

IntoTheHeaven 서비스를 이용하려면 회원가입 후 교회에 등록해야 합니다. 가입 후 교회를 검색하여 편입 요청을 보내면, 교회 관리자가 승인한 뒤 서비스를 이용할 수 있습니다.

## 주요 기능

### 회원가입

새로운 계정을 생성합니다.

1. 로그인 화면에서 **회원가입** 버튼을 누릅니다.
2. 다음 정보를 입력합니다.
   - **이름** (필수)
   - **성별** (필수)
   - **생년월일** (필수)
   - **전화번호** (필수) - 중복 확인 버튼을 눌러 이미 가입된 번호인지 확인합니다.
   - **이메일** (필수) - 중복 확인 후 인증 코드가 이메일로 발송됩니다. 수신한 인증 코드를 입력하여 이메일을 인증합니다.
   - **비밀번호** (필수) - 8자 이상으로 설정합니다.
   - **주소** - 우편번호 검색(다음 주소 API)을 통해 입력합니다.
3. 모든 항목을 입력한 뒤 **가입하기** 버튼을 누르면 가입이 완료됩니다.

### 로그인

가입한 이메일과 비밀번호로 로그인합니다.

1. 이메일과 비밀번호를 입력합니다.
2. **로그인** 버튼을 누릅니다.
3. 로그인 후 소속 교회에 따라 다음과 같이 동작합니다.
   - **소속 교회가 없는 경우**: 교회 등록 화면으로 이동합니다.
   - **소속 교회가 1개인 경우**: 해당 교회의 홈 화면으로 바로 진입합니다.
   - **소속 교회가 2개 이상인 경우**: 교회 선택 화면이 표시됩니다. 원하는 교회를 선택하여 진입합니다.

### 교회 등록 요청

소속 교회에 편입을 요청합니다.

1. 교회 검색창에서 교회 이름을 검색합니다.
2. 검색 결과에서 해당 교회를 선택합니다.
3. **가입 요청** 버튼을 누릅니다.
4. 교회 관리자가 요청을 승인하면 해당 교회의 서비스를 이용할 수 있습니다. 승인 전까지는 대기 상태입니다.

### 비밀번호 찾기

비밀번호를 잊어버린 경우 재설정할 수 있습니다.

1. 로그인 화면에서 **비밀번호 찾기**를 누릅니다.
2. 가입 시 사용한 **이메일**과 **이름**을 입력합니다.
3. 이메일로 발송된 **인증 코드**를 입력합니다.
4. **새 비밀번호**(8자 이상)를 설정하면 비밀번호가 변경됩니다.

## 알아두면 좋은 점

- 이메일은 로그인 ID로 사용되므로, 실제로 수신 가능한 이메일을 입력해야 합니다.
- 전화번호와 이메일은 이미 가입된 정보와 중복될 수 없습니다.
- 교회 편입 요청은 교회 관리자가 승인해야 완료됩니다. 승인까지 시간이 걸릴 수 있습니다.
