# 알림

> 시스템에서 보내는 알림을 확인합니다.

## 화면 설명

목회자 코멘트 등록, 소모임 정보 업데이트 등 서비스에서 발생하는 주요 알림을 받아볼 수 있습니다.

## 주요 기능

### 알림 목록

알림은 시간순으로 표시됩니다.

- **미읽음 알림**: 파란색 배경에 빨간 배지가 표시됩니다.
- **읽은 알림**: 회색 배경으로 표시됩니다.

### 알림 타입

다음과 같은 알림을 받을 수 있습니다.

- **목회자 코멘트 등록**: 관리자가 소그룹 모임에 코멘트를 남겼을 때
- **소모임 정보 업데이트**: 소속 소그룹의 정보가 변경되었을 때

### 알림 클릭

알림을 클릭하면 다음과 같이 동작합니다.

1. 해당 알림이 읽음 처리됩니다.
2. 알림과 관련된 페이지로 자동 이동합니다.
3. 알림 타입에 따라 해당 영역이 하이라이트됩니다.
   - **목회자 코멘트 등록**: 모임 상세 화면의 목회자 코멘트 영역으로 스크롤되고, 해당 영역이 반짝입니다.
   - **소모임 정보 업데이트**: 모임 상세 화면에서 나의 멤버 카드가 자동으로 펼쳐지고, 해당 카드로 스크롤된 후 테두리가 반짝입니다.

## 알아두면 좋은 점

- 홈 화면의 알림 아이콘에 미읽음 알림 수가 배지로 표시됩니다.
