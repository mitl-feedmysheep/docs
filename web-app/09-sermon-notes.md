# 설교노트

> 예배 중 들은 설교를 기록하고 다시 돌아볼 수 있는 개인 노트 기능입니다.

## 화면 설명

하단 탭 바의 **설교노트** 탭을 통해 접근합니다. 내가 작성한 설교 노트를 월별로 모아 확인하고, 새로운 노트를 작성하거나 기존 노트를 수정·삭제할 수 있습니다.

설교노트는 완전히 개인적인 기능으로, 다른 멤버와 공유되지 않습니다.

## 주요 기능

### 설교노트 목록

설교노트는 설교 날짜를 기준으로 **월별 그룹**으로 묶여 표시됩니다.

각 노트 카드에는 다음 정보가 표시됩니다.

- **설교 제목**
- **설교 날짜**
- **설교자** (입력한 경우)
- **예배 종류** (입력한 경우)

카드를 누르면 상세 화면으로 이동합니다.

### 설교노트 작성

목록 화면 우측 하단의 **+ 버튼**을 누르면 작성 화면으로 이동합니다.

입력 필드는 다음과 같습니다.

| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| 설교 제목 | 필수 | 최대 100자 |
| 설교 날짜 | 필수 | 날짜 선택 |
| 설교자 | 선택 | 목사님 이름, 최대 50자 |
| 예배 종류 | 선택 | 예: 주일 1부, 청년부 예배 등. 최대 50자. 이전에 입력한 값이 자동완성으로 제안됩니다 |
| 본문 말씀 | 선택 | 성경 구절, 최대 200자 |
| 나의 노트 | 필수 | 설교 내용 기록 |

### 설교노트 상세

노트의 전체 내용을 확인할 수 있습니다.

상단에는 제목, 날짜, 설교자, 예배 종류가 표시되고, 본문 말씀이 있으면 별도 카드로 강조 표시됩니다.

우측 상단의 아이콘으로 다음 작업을 할 수 있습니다.

- **수정** (연필 아이콘): 노트 수정 화면으로 이동합니다.
- **삭제** (휴지통 아이콘): 확인 다이얼로그가 표시되며, 확인하면 노트가 삭제됩니다.

### 설교노트 수정

상세 화면에서 수정 버튼을 누르면 기존 내용이 채워진 편집 화면으로 이동합니다. 작성 화면과 동일한 필드를 수정할 수 있습니다.

## 알아두면 좋은 점

- 설교노트는 교회나 소그룹과 무관하게 **개인 단위**로 관리됩니다.
- 다른 멤버에게 공유되지 않는 **완전히 개인적인 기록**입니다.
- 예배 종류 입력 시, 이전에 입력한 값들이 자동완성으로 제안되어 편리하게 재사용할 수 있습니다.
- 삭제된 노트는 복구할 수 없습니다.
