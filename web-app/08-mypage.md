# 마이페이지

> 내 정보를 확인하고 수정합니다.

## 화면 설명

내 프로필 정보를 확인하고 수정할 수 있습니다. 비밀번호 변경, 교회 전환, 로그아웃 기능을 제공합니다.

## 주요 기능

### 사용자 정보 카드

화면 상단에 내 프로필 정보가 표시됩니다.

- 아바타 이미지
- 이름
- 이메일

### 내 정보 수정

프로필 정보를 수정할 수 있습니다.

1. **내 정보 수정** 메뉴를 누릅니다.
2. 다음 항목을 수정할 수 있습니다.
   - 이름
   - 성별
   - 생년월일
   - 전화번호
   - 주소
   - 세례 여부 (세례 / 미세례 / 유아세례)
   - MBTI
   - 직업
3. 수정 후 **저장** 버튼을 누릅니다.

### 비밀번호 변경

비밀번호를 변경합니다.

1. **비밀번호 변경** 메뉴를 누릅니다.
2. **현재 비밀번호**를 입력합니다.
3. **새 비밀번호**(8자 이상)를 입력합니다.
4. 변경이 완료되면 보안을 위해 로그인 페이지로 자동 이동합니다. 새 비밀번호로 다시 로그인합니다.

### 교회 전환

여러 교회에 소속되어 있는 경우 교회를 전환할 수 있습니다.

1. **교회 전환** 메뉴를 누릅니다.
2. 소속 교회 목록에서 전환할 교회를 선택합니다.

### 로그아웃

**로그아웃** 버튼을 누르면 로그인 화면으로 이동합니다.

## 알아두면 좋은 점

- 비밀번호 변경 후에는 반드시 다시 로그인해야 합니다.
- 교회 전환은 2개 이상의 교회에 소속된 경우에만 표시됩니다.
