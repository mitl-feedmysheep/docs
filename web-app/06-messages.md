# 메시지

> 교회 멤버에게 메시지를 보내고 받을 수 있습니다.

## 화면 설명

멤버 간 1:1 메시지를 주고받는 기능입니다. 생일 축하 메시지와 일반 메시지를 구분하여 관리합니다.

## 주요 기능

### 탭 전환

상단 탭으로 메시지를 분류하여 봅니다.

- **받은 메시지**: 다른 멤버가 나에게 보낸 메시지
- **보낸 메시지**: 내가 다른 멤버에게 보낸 메시지

### 메시지 카드

각 메시지 카드에는 다음 정보가 표시됩니다.

- **타입 표시**: 생일 축하 메시지는 축하 이모지, 일반 메시지는 말풍선 이모지로 구분됩니다.
- **발신자/수신자 이름**
- **메시지 내용**
- **시간**: 상대 시간으로 표시됩니다 (예: "3시간 전", "어제")
- **미읽음 배지**: 아직 읽지 않은 메시지에 표시됩니다.

### 읽음 처리

메시지를 클릭하면 자동으로 읽음 처리됩니다.

## 알아두면 좋은 점

- 메시지는 홈 화면의 교적부 검색이나 생일자 섹션에서 보낼 수 있습니다.
- 홈 화면의 메시지 아이콘에 미읽음 메시지 수가 배지로 표시됩니다.
